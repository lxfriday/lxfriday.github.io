1 英文字母解释

  1. HTTP (HyperText Transfer Protocol) 超文本传输协议
  2. URL (Uniform Resource Locator) 统一资源定位符
  3. FTP (File Transfer Protocol) 文件传输协议
  4. DNS (Domain Name Service) 域名服务
  5. TCP (Transmission Control Protocol) 传输控制协议
  6. UDP (User Data Protocol) 用户数据报协议
  7. NIC (Network Interface Card) 网络适配器（网卡）
  8. IP (Internet Protocol) 网络协议
  9. MAC (Media Access Control Address) 媒体访问控制地址
  10. ARP (Address Resolution Protocol) 地址解析协议
  11. RFC (Request for Comments) 征求修正意见书
  12. REST (Representational State Transfer) 表征状态转移
  13. TLS (Transport Layer Security) 传输层安全
  14. SSL (Secure Socket Layer) 安全套接层
  15. MIME (Multipurpose Internet Mail Extensions) 多用途因特网邮件扩展
  16. WebDAV (Web-based Distributed Authoring and Versioning) 基于万维网的分布式创作和版本控制
  17. CRC (Cyclic Redundancy Check) 循环冗余校验
  18. Dos 攻击 (Denial of Service) 拒绝服务攻击
  19. MITM (Man-in-the-Middle) 中间人攻击，请求或响应在传输图中，遭攻击者拦截并篡改内容的攻击成为中间人攻击
  20. EV SSL 整数(Extended Validation SSL Certificate)
  21. CRL (Certificate Revocation List) 整数吊销机制
  22. IETF (Internet Engineering Task Force) 互联网工程任务组
  23. XSS (Cross-Site Scripting) 跨站脚本攻击
  24. CSRF (Cross-site Request Forgery) 跨站请求伪造攻击
